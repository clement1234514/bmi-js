# BMI Rechner — HTML / CSS / JavaScript

Dieses Projekt ist eine einfache Webanwendung, mit der man den BMI direkt im Browser berechnen kann, ohne Backend.  
Es ist vollständig kompatibel mit **GitHub Pages**, wodurch die Anwendung online getestet werden kann.

---

# Ziel des Projekts

- Eine interaktive Web-Minianwendung erstellen
- Das DOM mit JavaScript bearbeiten
- Ein einfaches Design mit CSS anwenden
- Eine statische Anwendung auf GitHub Pages bereitstellen
- Ein klares und professionelles Projekt in einem Portfolio präsentieren

##  Funktionsweise

Der Benutzer gibt Folgendes ein:
- sein Gewicht (kg)
- seine Körpergröße (cm)

JavaScript berechnet den BMI anhand der folgenden Formel:

\[
BMI = \frac{Gewicht}{(größe\_in\_m)^2}
\]

Anschließend werden angezeigt:
- der BMI-Wert
- eine Kategorie (normal, übergewichtig usw.)
